|------|------|

| **状态** | 🟢 已解决 |
| **发现时间** | 2026-07-07 |
| **发现场景** | 分析 PRD v2.0 执行可行性 |
| **问题描述** | PRD 中 3 个"内嵌组件"（业务核实确认、归档入库、跟踪验收）格式与独立页面完全一致，但 skill 模板体系中无"内嵌组件/弹窗/抽屉"页型，AI 可能误按独立页面生成 |

## 使用说明

**AI 更新时机**：
1. 分析阶段发现新问题 → 添加新条目
2. 解决问题时 → 修改状态为"已解决"，填写解决记录
3. 测试验证机制时 → 更新"待验证"状态为"已验证"或"验证失败"

**PM 更新时机**：
1. 提供补充信息解决问题时 → 在对应问题下添加备注
2. 确认问题解决时 → 修改状态
3. 发现新问题时 → 添加新条目或告知 AI 添加
