# 绿星智审 — 应用模块划分与编码

> **编码规则：** 一级标识 = 2位拼音缩写，二级标识 = 2位拼音缩写，拼接为完整编码
>
> **阶段说明：** 现阶段聚焦4个一级模块，每模块按原型页面内容拆分1~2个二级模块

## 编码速查

|      编码      | 模块路径             | 对应原型页面                                                                                                                                          |
| :------------: | :------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------- |
| **FXKB** | 风险管理 → 看板场景 | 风险驾驶舱（风险总览看板、统计报表）、风险监督场景（场景卡片与风险点穿透）                                                                            |
| **FXSC** | 风险管理 → 事件池   | 风险事件池（工作台风险管理清单、事件详情、事件录入、等级调整审核）、风险事件导入                                                                      |
| **RWGL** | 任务中心 → 管理监控 | 任务驾驶舱、任务管理列表、任务创建、任务详情、我的任务                                                                                                |
| **RWZG** | 任务中心 → 整改闭环 | 整改方案填报、任务验收审核、超期督办、签收确认、业务核实确认、跟踪验收、归档入库                                                                      |
| **MXGZ** | 模型中心 → 规则引擎 | 规则配置管理、规则配置编辑、规则命中日志、规则测试验证、风险锁配置                                                                                    |
| **MXCK** | 模型中心 → 模型仓库 | 采购业务健康度（展示/编辑/详情）、供应商评分卡（展示/编辑）、指标管理、指标新增、评估结果查询、评估结果详情、智能派单（派单规则配置/结果查询/知识库） |
| **XTYH** | 系统管理 → 用户权限 | 用户管理、角色权限配置、组织架构维护                                                                                                                  |
| **XTPZ** | 系统管理 → 系统配置 | 流程配置、消息通知配置、系统参数配置、操作日志、控制点阈值配置                                                                                        |
