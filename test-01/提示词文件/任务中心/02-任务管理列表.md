# 任务管理列表

- **所属模块：** 任务中心 → 管理监控（RWGL）
- **优先级：** P0
- **原型参考：** `任务中心/2.任务管理列表.html`
- **页面类型：** 列表/查询

## 功能说明

全部任务统一列表，管理员视角。支持按状态、责任单位、期限等多维度筛选和批量操作。

## 关键数据字段

- 任务编号、任务标题
- 关联风险事件编号
- 责任单位、责任人
- 任务状态（待签收/处理中/待验收/已完成/已归档）
- 整改期限、创建时间
- 优先级

## 交互操作

- 多条件筛选（状态、责任单位、时间范围、关键字搜索）
- 批量操作（批量指派、批量督办）
- 点击任务行跳转至任务详情
- 新建任务按钮 → 跳转任务创建
- 列表分页
