# 任务验收审核

- **所属模块：** 任务中心 → 整改闭环（RWZG）
- **优先级：** P0
- **原型参考：** `任务中心/7.任务验收审核.html`
- **页面类型：** 审核/审批

## 功能说明

验收人对整改结果进行审核，支持通过/驳回操作。

## 关键展示内容

- 任务基本信息摘要
- 整改方案及执行情况
- 整改证据/附件
- 处置全流程时间线

## 交互操作

- 通过按钮 → 验收通过，任务进入归档流程
- 驳回按钮 → 需填写驳回原因，任务退回至责任人重新整改
- 查看整改详情

## 业务上下文

任务闭环链路的关键审批节点。驳回后任务回到整改方案填报环节。
