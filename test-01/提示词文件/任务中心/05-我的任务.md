# 我的任务

- **所属模块：** 任务中心 → 管理监控（RWGL）
- **优先级：** P0
- **原型参考：** `任务中心/5.我的任务.html`
- **页面类型：** 列表/查询

## 功能说明

当前登录用户的待办/已办任务列表。展示指派给自己的所有任务，支持查询和快速处理入口。

## 关键数据字段

- 任务编号、任务标题
- 关联风险事件编号
- 任务状态（待签收/处理中/待验收/已完成）
- 整改期限
- 创建时间
- 发起人

## 交互操作

- 切换待办/已办 tab
- 按状态、时间筛选
- 关键字搜索
- 点击任务行进入任务详情
- 快速处理入口（根据状态显示：签收/填报整改方案/提交验收等）
