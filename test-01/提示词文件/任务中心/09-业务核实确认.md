# 业务核实确认（内嵌组件）

- **所属模块：** 任务中心 → 整改闭环（RWZG）
- **优先级：** P0
- **原型参考：** `任务中心/内嵌-业务核实确认.html`
- **页面类型：** 内嵌弹窗/组件

## 功能说明

核实风险实际情况，确认业务数据是否存在异常。通常是任务处置流程中的一个环节，以内嵌弹窗形式出现在任务详情页中。

## 关键内容

- 任务基本信息摘要
- 风险事件描述
- 业务数据快照（原始数据 vs 检测结果）
- 核实结论选项

## 交互操作

- 确认异常 → 核实确认存在风险，任务继续流转至整改环节
- 误报 → 标记为误报，说明原因，任务可能关闭或退回
- 核实意见（文本输入）
- 提交按钮

## 业务上下文

任务闭环链路节点：签收确认 → **业务核实确认** → 整改方案填报 → 跟踪验收 → 归档入库。
