# 风险登记录入

- **所属模块：** 风险管理 → 事件池（FXSC）
- **优先级：** P0
- **原型参考：** `风险管理/风险事件池/9.风险登记录入.html`
- **页面类型：** 表单/录入

## 功能说明

手工录入风险事件（作为模型自动生成的补充）。支持填写事件基本信息、关联模型、上传证据附件。

## 关键表单字段

- 事件标题
- 来源类型（手工录入/模型命中/外部导入）
- 关联模型（可选）
- 风险等级（高/中/低）
- 业务场景（准入/寻源/合同/结算/供应商/综合）
- 涉及供应商
- 涉及金额
- 事件描述
- 证据附件（多文件上传）

## 交互操作

- 提交 → 创建风险事件，进入事件池列表
- 暂存草稿
- 取消 → 返回列表
