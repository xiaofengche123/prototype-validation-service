# 风险事件详情

- **所属模块：** 风险管理 → 事件池（FXSC）
- **优先级：** P0
- **原型参考：** `风险管理/风险事件池/风险事件详情.html`
- **页面类型：** 详情展示

## 功能说明

展示风险事件完整信息，包括基本信息、模型命中详情、关联数据快照、处置历史时间线。

## 关键展示内容

- 事件基本信息（编号、标题、等级、状态、来源）
- 来源模型信息（如来自模型命中，显示命中规则、命中时间、置信度）
- 关联数据快照（触发时的业务数据上下文）
- 涉及供应商、涉及金额
- 处置历史时间线（确认→转任务→处置→关闭，每节点时间和操作人）
- 关联任务列表（由该事件创建的整改任务）

## 交互操作

- **转为任务按钮（PRD 新增）** → 跳转任务创建页，自动带入事件编号、事件描述、涉及供应商
- 等级调整 → 跳转等级调整审核
- 关闭事件
- 返回列表

## PRD 备注

新增「转为任务」按钮，风控专员确认事件后可一键转任务。
