# 采购业务健康度编辑

- **所属模块：** 模型中心 → 模型仓库（MXCK）
- **优先级：** P0
- **原型参考：** `模型中心/模型库/采购业务健康度编辑.html`
- **页面类型：** 表单/编辑

## 功能说明

编辑采购业务健康度模型的指标权重和阈值配置。

## 关键表单字段

每个评估维度可配置：
- 维度名称（质量/交付/价格/服务/合规等）
- 权重（百分比，各维度权重之和 = 100%）
- 关联指标（从指标库选择，可多选）
- 各指标阈值范围
- 评分规则（如何从指标值计算维度得分）

## 交互操作

- 拖拽调整维度权重
- 添加/删除维度
- 为维度关联指标
- 预览评分 → 用样例数据验证配置
- 保存
- 取消 → 返回健康度看板
