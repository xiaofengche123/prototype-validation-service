# 供应商评分卡编辑

- **所属模块：** 模型中心 → 模型仓库（MXCK）
- **优先级：** P0
- **原型参考：** `模型中心/模型库/供应商评分卡编辑.html`
- **页面类型：** 表单/编辑

## 功能说明

编辑供应商评分卡模型的评分维度和权重配置。

## 关键表单字段

每个评分维度可配置：
- 维度名称（质量/交付/价格/服务）
- 权重（百分比，各维度权重之和 = 100%）
- 关联指标（从指标库选择）
- 评分规则（各指标如何映射到维度得分）
- 等级划分标准（优秀≥90/良好≥75/一般≥60/较差<60）

## 交互操作

- 拖拽调整维度权重
- 添加/删除维度
- 为维度关联指标
- 预览评分 → 选取供应商验证评分结果
- 保存
- 取消 → 返回供应商评分卡查看
