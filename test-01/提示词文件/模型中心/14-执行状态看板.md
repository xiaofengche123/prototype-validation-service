# 执行状态看板

- **所属模块：** 模型中心 → 执行监控（暂无独立编码，暂归 MXGZ/MXCK）
- **优先级：** P0
- **原型参考：** `模型中心/执行监控/执行状态看板.html`
- **页面类型：** 监控看板

## 功能说明

模型执行状态监控面板，展示各模型运行状态（正常/异常/未执行）、最近执行时间、执行耗时。用于系统管理员和模型管理员监控所有模型的执行健康状况。

## 关键展示内容

- 模型总数、运行中数量、异常数量、未执行数量（统计卡片）
- 模型执行状态列表/卡片，每条展示：
  - 模型名称
  - 运行状态（正常/异常/未执行，颜色标识）
  - 最近执行时间
  - 执行耗时
  - 下次计划执行时间
  - 最近命中次数

## 交互操作

- 按状态筛选（正常/异常/未执行）
- 点击模型 → 跳转执行结果查询（该模型的执行历史）
- 手动触发执行按钮（针对未执行/异常模型）
- 刷新
