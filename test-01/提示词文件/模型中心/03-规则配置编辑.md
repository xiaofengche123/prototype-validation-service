# 规则配置编辑

- **所属模块：** 模型中心 → 规则引擎（MXGZ）
- **优先级：** P0
- **原型参考：** `模型中心/规则引擎/规则配置编辑.html`
- **页面类型：** 表单/编辑器

## 功能说明

规则条件编辑器，支持配置规则名称、所属场景、触发条件（字段+运算符+阈值）、风险等级、处置方式等。

## 关键表单字段

- 规则名称
- 所属业务场景（下拉选择）
- 规则描述
- 触发条件配置（多条件组合）：
  - 数据字段（选择）
  - 运算符（等于/大于/小于/包含/不为空等）
  - 阈值
  - 条件逻辑（AND/OR 组合）
- 命中后风险等级（高/中/低）
- 置信度阈值
- 是否自动生成风险事件
- 处置方式建议
- 启用状态

## 交互操作

- 添加/删除条件行
- 条件组合（AND/OR 切换）
- 保存/保存并启用
- 测试验证按钮 → 用历史数据测试规则
- 取消 → 返回规则配置管理列表

## PRD 备注

自动生成阈值可在规则配置中设定，模型执行命中后根据严重程度和置信度自动生成风险事件。
