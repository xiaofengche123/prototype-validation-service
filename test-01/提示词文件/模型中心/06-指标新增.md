# 指标新增

- **所属模块：** 模型中心 → 模型仓库（MXCK）
- **优先级：** P0
- **原型参考：** `模型中心/模型库/指标新增.html`
- **页面类型：** 表单/录入

## 功能说明

新增指标定义，配置指标名称、计算口径、数据来源、阈值范围等。

## 关键表单字段

- 指标名称
- 所属模型（下拉选择：采购业务健康度/供应商评分卡/其他）
- 计算口径/公式
- 数据来源（选择数据表/字段）
- 阈值范围（最小值-最大值）
- 阈值方向（越高越好/越低越好/区间合理）
- 权重（用于综合评分计算）
- 指标说明

## 交互操作

- 提交 → 指标创建，返回指标管理列表
- 取消 → 返回指标管理列表
- 测试计算 → 使用样例数据预览计算结果
