# 模板 · 方案对比页

> 内部页型：方案对比分析页。多个方案放一起比、选优采用时用。
> 三段结构：① PM 可见业务要素 ② 内置约束（隐藏，含 IR）③ 元追问清单。

---

## ① 业务要素（业务语言）

- 页面中文名、菜单分组
- 顶部说明条：对比规则
- 上方：左侧方案散点图/趋势图 + 右侧方案列表（可勾选**最多几个**方案）
- 下方：方案详细对比表（第一列是对比维度，后续列是选中的方案，**推荐方案高亮**）
- 底部操作：返回、采用当前方案

## ② 内置约束（PM 不可见 · 生成时落实）

遵循 `../../prototype-site/SKILL.md` 与 `interaction-ir.md`：

- 散点/趋势图用占位或 CSS 简易图形；点图中点位或方案列表可**选中方案**。
- 选择**最多 N 个**，超过时提示（N 由 PM 给，默认 3）。
- 对比表第一列为维度，后续列为选中方案；推荐方案高亮；点方案列可切换"当前采用"。
- 返回 `data-nav="<返回页编码>"`；采用 `data-action="adopt"`。
- **底部操作栏固定**（`has-fixed-actions` + `.fixed-bottom-bar`）。
- 无页面级横向滚动（对比表过宽只在 wrapper 内横滚）；上方左右区等高。
- 菜单幂等注册。

## ③ 元追问清单（业务语言，少量多次）

1. 对比的是什么方案？页面叫什么、放哪个分组？
2. 按哪些**维度**对比（成本/质量/工期…）？
3. 最多允许同时比**几个**方案？（默认 3）
4. 上方想用什么图来看方案分布（散点/趋势）？
5. 怎么算**推荐方案**？要不要"采用"按钮？

> 对比维度 + 最多个数是关键；推荐规则缺则标【补位】。
