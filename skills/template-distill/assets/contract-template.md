## §2 外壳规格

> 填写前先判定布局模式。两种模式二选一，另一种填"不适用"。
> - **侧边栏模式**：填写侧栏相关字段
> - **顶导航模式**：侧栏字段填"不适用—顶导航"，改填导航栏字段

| 区域 | 规格 | 出处 |
| --- | --- | --- |
| **布局模式** | `侧边栏` 或 `顶导航` | |
| 顶栏 | 高度 px、品牌区内容、右侧功能区 | |
| 侧栏（仅侧边栏模式）| 宽度 px、折叠宽度 px、菜单项高度 px、分组样式；顶导航模式填"不适用" | |
| 导航栏（仅顶导航模式）| 高度 px、背景色、菜单项字号/间距、hover 样式、active 表达；侧边栏模式填"不适用" | |
| 主容器（仅顶导航模式）| 最大宽度、居中方式、水平 padding；侧边栏模式填"不适用" | |
| 页脚（仅顶导航模式）| 背景色、高度/padding、内容；侧边栏模式填"不适用" | |
| 多页签栏 | 高度 px、页签形态；溢出滚动：禁原生横向滚动条（方向按钮 / AntD nav 操作区） | |
| 内容区 | 内边距 px、背景色 | |
| 响应式断点 | 断点 px、该断点下隐藏/显示哪些区域 | |

> 外壳规格同时约束原型 `index.html` 与 Vue `BasicLayout.vue`，两侧观感必须可并排比对。
