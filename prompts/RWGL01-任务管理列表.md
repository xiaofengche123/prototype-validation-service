## 业务内容结构

### 1. 查询区（8个字段）
| 字段 | 类型 | 说明 |
|------|------|------|
| 任务编号 | 文本输入 | 请输入任务编号 |
| 任务标题 | 文本输入 | 请输入任务标题 |
| 任务类型 | 下拉选择 | 风险处置/专项审计/日常巡检/督办整改 |
| 当前状态 | 下拉选择 | 全部状态/待核实/待派发/待签收/处理中/待验收/已完成/已退回 |
| 负责人 | 文本输入 | 请输入负责人 |
| 责任部门 | 下拉选择 | 采购部/财务部/质量管理部/法务部/审计部/仓储部 |
| 创建时间 | 日期范围 | 2024-10-01 至 2024-12-31 |
| 截止时间 | 日期范围 | - |

### 2. 状态页签（8个）
全部(186) / 待核实(8) / 待派发(23) / 待签收(15) / 处理中(89) / 待验收(42) / 已完成(14) / 已退回(3)

### 3. 表格列（13列）
| 列 | 说明 |
|----|------|
| 复选框 | 批量选择 |
| 序号 | - |
| 任务编号 | 可点击链接，跳转任务详情 |
| 任务标题 | 左对齐，超长省略 |
| 任务类型 | - |
| 关联风险 | 数字 |
| 负责人 | - |
| 责任部门 | - |
| 创建时间 | - |
| 计划截止 | 超期标红 |
| 当前状态 | 状态标签（待核实/待派发/待签收/处理中/待验收/已完成/已退回） |
| 派单依据 | 规则引擎推荐依据 tooltip |
| 操作 | 查看/派发/删除/催办/跟踪/验收/归档/重新派发 |

### 4. 批量操作栏
- 已选择 N 项
- 批量派发 / 批量删除

### 5. 分页器
- 共 186 条
- 每页条数选择（10/20/50）
- 页码导航（1-5页）
- 前往 X 页

### 6. 顶部操作按钮
- 【创建任务】按钮 → 跳转任务创建页
- 【批量导出】按钮
- 【批量派发】按钮

---

## 交互
- 状态页签点击过滤表格
- 表格行复选框触发批量操作栏显示
- 任务编号点击跳转任务详情（RWMX01）
- 操作按钮：查看/派发/删除/催办/跟踪/验收/归档/重新派发
- 创建任务按钮跳转 RWCJ01

---

## 状态key
- 待核实(st-draft灰色) / 待派发(st-draft灰色) / 待签收(st-pending蓝色)
- 处理中(st-process紫色) / 待验收(st-check橙色) / 已完成(st-done绿色) / 已退回(st-reject红色)
