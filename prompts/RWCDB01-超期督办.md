列表查询页（督办专用）

## 页面中文名
超期督办

## 分组
任务管理

## 旧HTML提取摘要
从 `原型/任务中心/8.超期督办.html` 提取。旧页面包含完整外壳，本次只保留业务内容区。

## 业务内容结构

### 1. 页头
- 标题：超期督办
- 操作按钮：返回任务列表

### 2. 预警统计卡（3张渐变卡）
| 颜色 | 图标 | 数值 | 标签 |
|------|------|------|------|
| 红色渐变 | 警铃 | 12 | 已超期 |
| 橙色渐变 | 时钟 | 5 | 今日到期 |
| 黄色渐变 | 警告 | 8 | 3日内到期 |

### 3. 查询区（6个字段）
| 字段 | 类型 |
|------|------|
| 任务编号 | 文本输入 |
| 任务标题 | 文本输入 |
| 责任人 | 文本输入 |
| 责任部门 | 下拉选择（采购部/财务部/质量管理部/法务部/审计部/仓储部） |
| 超期天数起 | 数字输入 |
| 超期天数止 | 数字输入 |

### 4. 状态页签（5个）
全部(25) / 已超期(12) / 今日到期(5) / 3日内到期(5) / 7日内到期(3)

### 5. 表格列（12列）
| 列 | 说明 |
|----|------|
| 复选框 | 批量选择 |
| 序号 | - |
| 任务编号 | 可点击链接，跳转任务详情(RWMX01) |
| 任务标题 | 左对齐，超长省略 |
| 责任人 | - |
| 责任部门 | - |
| 计划截止 | - |
| 超期天数 | 红色加粗（超期）/ 橙色（今日到期）/ 普通（即将到期） |
| 预警等级 | 红色预警/橙色预警/黄色预警（带圆点标签） |
| 催办次数 | 数字 |
| 当前状态 | - |
| 操作 | 催办/更新/切换责任人/查看 |

### 6. 示例数据（8条）
- 已超期 3 条（+5天/+3天/+17天，红色预警）
- 今日到期 2 条（橙色预警）
- 3日内到期 2 条（剩3天/剩2天，黄色预警）
- 7日内到期 1 条（剩7天，黄色预警）

### 7. 弹窗
- **更新任务弹窗**：任务编号(只读)、任务标题(只读)、截止日期、优先级、任务描述、更新说明
- **切换责任人弹窗**：任务编号(只读)、原责任人(只读)、新责任人(下拉)、抄送人标签(可增删)、交接说明

### 8. 分页器
- 共 25 条 / 页码导航 / 前往 X 页

---

## 交互
- 状态页签点击过滤表格
- 催办按钮：confirm 确认后 toast 提示
- 更新按钮：打开更新弹窗
- 切换责任人按钮：打开切换责任人弹窗
- 任务编号点击跳转任务详情（RWMX01）
- 抄送人标签可删除、可添加（随机从预设列表选）

---

## 状态key
- 预警等级：红色预警(warn-red)/橙色预警(warn-orange)/黄色预警(warn-yellow)
- 超期状态：已超期(overdue)/今日到期(today)/3日内到期(d3)/7日内到期(d7)
- 催办次数：0-3次
