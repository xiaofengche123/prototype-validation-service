## 业务内容结构

### 1. 查询区（8个字段）
| 字段 | 类型 | 说明 |
|------|------|------|
| 风险编号 | 文本输入 | 请输入风险编号 |
| 风险标题 | 文本输入 | 请输入风险标题 |
| 风险分类 | 下拉选择 | 计划管理风险/供应商管理风险/招投标风险/合同管理风险/价格与结算风险/质量与库存风险/廉洁合规风险 |
| 风险等级 | 下拉选择 | 全部等级/红色-重大/橙色-较大/黄色-一般/蓝色-关注 |
| 发现来源 | 下拉选择 | 系统监测/审计移送/业务报告/举报线索/外部舆情 |
| 责任部门 | 下拉选择 | 采购部/财务部/质量管理部/法务部/审计部 |
| 发现时间 | 日期范围 | 2024-10-01 至 2024-12-31 |
| 当前状态 | 下拉选择 | 全部状态/待签收/处置中/待验收/已办结 |

### 2. 状态页签（5个）
全部(128) / 待签收(15) / 处置中(42) / 待验收(28) / 已办结(43)

### 3. 表格列（12列）
| 列 | 说明 |
|----|------|
| 复选框 | 批量选择，全选/反选 |
| 序号 | - |
| 风险编号 | 可点击链接，蓝色 |
| 风险标题 | 左对齐，超长省略，max-width:200px |
| 风险分类 | - |
| 风险等级 | 等级标签：红色-重大/橙色-较大/黄色-一般/蓝色-关注，带色点前缀 |
| 涉及金额(万元) | 数值，无金额显示"--" |
| 责任部门 | - |
| 责任人 | - |
| 发现时间 | 日期时间格式 |
| 当前状态 | 状态标签：待签收(蓝色)/处置中(紫色)/待验收(橙色)/已办结(绿色)，带色点前缀 |
| 操作 | 签收/填报进度/提交验收/归档（按状态不同） |

### 4. 批量操作栏
- 已选择 N 项
- 批量签收 / 批量派单
- 有选中行时显示，无选中时隐藏

### 5. 分页器
- 共 128 条
- 每页条数选择（10/20/50），默认20条/页
- 页码导航（1-5页）
- 前往 X 页

### 6. 顶部操作按钮
- 【批量导出】按钮
- 【批量签收】按钮
- 【+ 新增录入】按钮 → 跳转风险登记录入页

---

## 交互
- 状态页签点击过滤表格，点击后重新计算序号
- 表格行复选框触发批量操作栏显示，显示已选数量
- 全选/反选控制所有行复选框
- 风险编号点击可跳转详情（链接占位）
- 查询/重置按钮
- 操作按钮根据状态动态显示：
  - 待签收 → 签收
  - 处置中 → 填报进度
  - 待验收 → 提交验收
  - 已办结 → 归档

---

## 状态key
- 待签收(st-wait蓝色) / 处置中(st-process紫色) / 待验收(st-check橙色) / 已办结(st-done绿色)
- 风险等级：red(红色-重大) / orange(橙色-较大) / yellow(黄色-一般) / blue(蓝色-关注)
