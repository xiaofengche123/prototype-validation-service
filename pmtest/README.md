# pmtest · 产品经理验收测试目录

本目录用于在交付 `prototype-guide` + `prototype-site` 给产品经理前，进行 Kimi 端到端手动测试。

## 文件说明

| 文件 | 用途 |
|---|---|
| `操作指南.md` | 一步步教你如何启动 Kimi、扮演 PM、录屏截图、验证产物。 |
| `测试结果模板.md` | 测试过程中/测试后填写，逐条打勾并记录问题。 |
| `screen-recordings/` | 存放完整测试录屏。 |
| `screenshots/` | 存放关键步骤截图。 |

## 测试来源

- 测试案例：`../skills/prototype-guide/测试与交付/05-手动测试案例-供应商证照(Kimi).md`
- 对照案例（Claude 版）：`../skills/prototype-guide/测试与交付/04-手动测试案例-供应商证照(Claude).md`

## 快速开始

1. 读 `操作指南.md`。
2. 按指南启动 Kimi 并运行测试。
3. 把录屏和截图分别放入 `screen-recordings/` 和 `screenshots/`。
4. 填写 `测试结果模板.md`。
